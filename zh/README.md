## 浏览器支持 

1. ie7+ ,由于产品需要，仍然要照顾到那些中国特色的低版本的ie用户，所以目前版本我们需要保证支持了ie7+. (事实上，ie6也在目前版本的支持范围内)

### 指南命名约定

* __Component__  指代所有Regular和继承自Regular的组件
* __component(小写)__  指代所有__Component__的实例
